# Kamel Ride Analytics Dashboard

This is a small Next.js + TypeScript dashboard with mock analytics data for the KamelRide product engineer role.

## What You Need

- Node.js `20` or newer
- `npm`

## Setup

1. Open a terminal in this folder.
2. Install packages:

```bash
npm install
```

3. Start the app:

```bash
npm run dev
```

4. Open this in your browser:

```text
http://localhost:3000
```

## If You Want To Check Production Build

```bash
npm run build
```

## Notes

- The dashboard uses mock data by default.
- There is a simple event API at `/api/events`.
- The main dashboard UI is in `src/components/dashboard.tsx`.
